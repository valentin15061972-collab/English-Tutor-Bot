# English-Tutor-Bot - это телеграм бот для изучения английского языка

## Инструменты и доп.материалы
1. Python 
2. IDE(GIGA IDE).
3. GIT + GitHub.
4. PostgreSQL
5. DBeaver
6. SQLAlchemy.ORM
7. pyTelegramBotAPI.
8. Статья по разработке бота в Telegram.
9. Инструкция по созданию бота в Telegram.
10. Подготовка к работе над проектом(видео).
11. Основа бота - код из модуля main.py.

## Возможности
- Перевод слов (английский <-> русский)
- Кнопки с четырьмя вариантами ответа (один правильный)
- Добавление слов в словарь
- Удаление слов из словаря
